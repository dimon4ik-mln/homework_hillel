NOTE:
- XPath supports text() and complex expressions.
- CSS selectors DO NOT support text(), so the "text()" requirement applies only to XPath.
- A few locators below are based on the public page-object structure for the related QAuto training site and common DOM patterns of this project. Before сдача, quickly verify them in DevTools on your side.

========================
25 XPATH LOCATORS
========================
1. Home menu link
//a[text()="Home"]

2. About button/link by text
//*[self::a or self::button][text()="About"]

3. Contacts button/link by text
//*[self::a or self::button][text()="Contacts"]

4. Sign In button by exact text
//button[text()="Sign In"]

5. Sign Up button by exact text
//button[text()="Sign Up"]

6. Guest login button by exact text
//button[text()="Guest log in"]

7. Main hero title by partial text
//h1[contains(text(),"Do more")]

8. Contacts section header
//h2[text()="Contacts"]

9. Footer link to Facebook by href attribute
//a[contains(@href,"facebook")]

10. Footer link to Telegram by href attribute
//a[contains(@href,"telegram")]

11. Sign in email input by name attribute
//input[@name="email"]

12. Sign in password input by id
//*[@id="signinPassword"]

13. Sign in modal title
//div[contains(@class,"modal-content")]//h4[text()="Log in"]

14. Login submit button inside sign in modal
//div[contains(@class,"modal-content")]//button[text()="Login"]

15. Registration name input by id
//*[@id="signupName"]

16. Registration last name input by id
//*[@id="signupLastName"]

17. Registration email input by id
//*[@id="signupEmail"]

18. Registration password input by id
//*[@id="signupPassword"]

19. Registration repeat password input by id
//*[@id="signupRepeatPassword"]

20. Register submit button inside modal
//div[contains(@class,"modal-content")]//button[text()="Register"]

21. Garage nav link by text
//a[text()="Garage"]

22. Fuel expenses nav link by text
//a[text()="Fuel expenses"]

23. Add car button on Garage page by class and text
//button[contains(@class,"btn-primary") and text()="Add car"]

24. Brand select in Add car modal by id
//*[@id="addCarBrand"]

25. Complex locator: Add button inside Add car modal footer
//div[contains(@class,"modal-content")]//div[contains(@class,"modal-footer")]//button[text()="Add"]
